API Documentation

Get Floor Price by Collection

Endpoint: GET /floor-price/:collection

Description: Get the floor price for a collection.

Parameters:

  • collection (string): The name of the collection.

Example: GET /floor-price/mycollection

Get Asset by ID

Endpoint: GET /asset/:assetID

Description: Get asset information by its ID.

Parameters:

  • assetID (string): The ID of the asset.

Response:

  • Success (200):
    • JSON object with the asset information.
  • Error (404):
    • JSON object with the error message if the asset is not found.

Example:
Request: GET /asset/123456
Response: 

{
  "data": {
    "collection": "Asset Collection Name"
  }
}

Get Rewards to Collect

Endpoint: POST /rewards-to-collect/

Description: Get the rewards to collect for a wallet address and asset ID.

Request Body:

  • walletAddress (string): The wallet address.
  • assetId (string): The asset ID.
  • contract (string): The contract information.
  • isToken (boolean): Whether it's a token.
  • isTestNet (boolean): Optional. Whether it's a testnet.

Example: POST /rewards-to-collect/
Request Body: 

{
  "walletAddress": "0x1234567890",
  "assetId": "123",
  "contract": "contract info",
  "isToken": true,
  "isTestNet": false
}

Get History of Items Collected

Endpoint: GET /history/:walletAddress/:network

Description: Get the history of items collected by a wallet address.

Parameters:

  • walletAddress (string): The wallet address.
  • network (string): The network (mainnet or testnet).

Example: GET /history/0x1234567890/mainnet
